# 2D 平台跳跃游戏 (Godot Engine)

一个完整的 2D 平台跳跃游戏，展示 Godot 引擎的 2D 开发能力。

## 核心特性
- **平台跳跃**：精准的跳跃手感和物理反馈
- **翻滚机制**：带无敌帧的闪避动作
- **Boss 战**：拥有瞬移技能的强力 Boss
- **关卡传送**：击败 Boss 后解锁传送门，进入下一关
- **隐藏墙**：探索隐藏区域获取额外奖励

## 操作指南
- W / ↑：跳跃
- A / ←：向左
- D / →：向右
- C / Shift：翻滚

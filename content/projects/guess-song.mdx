# 🎶 猜歌名挑战

一个有趣的音乐猜歌游戏，使用你自己的本地音乐库。

## 核心功能
- **自定义曲库**：完全使用本地音乐文件夹作为游戏曲库
- **多种难度**：从整首歌到几秒片段，适合各种玩家
- **智能自动补全**：根据曲库歌名提供智能提示
- **实时计分系统**：根据表现实时增减分数
- **紧张倒计时**：20 秒倒计时进度条让游戏更刺激

## 技术亮点
- 使用 C++ 和 Qt 框架开发
- 自动解析音乐文件元数据
- 跨平台桌面应用

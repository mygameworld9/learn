# GenUI - AI-Powered UI Generator

使用自然语言描述，让 AI 为你生成可交互的 UI 界面。

## 核心功能
- **自然语言生成**：描述你想要的界面，AI 自动生成代码
- **实时预览**：生成后即刻看到可交互界面
- **Gemini 驱动**：基于 Google Gemini API 的强大理解能力

## 技术栈
- **前端**：React / Next.js
- **后端**：Node.js
- **AI**：Gemini API

## 快速开始
```bash
npm install
npm run dev
```

设置 `GEMINI_API_KEY` 环境变量后启动应用。
